# Alien invasion game

This is a simple 2D shooter created by PyGame.

How to execute:
     1. execute run.sh script or "python3 main.py"

Requirement:
     1. pygame==1.9.6


How to run:
1. checkout the project
2. cd <checkout directory>
3. virtualenv venv
4. source venv/bin/activate
5. pip install -r requirements.txt
6. ./run.sh
